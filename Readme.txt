###########################   READ ME   ########################

1. Install the following library packages: 
    MASS
    klaR
    caret
    neuralnet
    rpart
    rpart.plot
    e1071
    mlbench

2. All the classifiers code including preprocessing data code is in the single R file - MLAssignment.R

3. Execute the file MLAssignment.R - it will exceute code for all the classifiers.
   For Perceptron classifier execution Kindly comment the 'nn' variable in neural nets    and uncomment the nn variable in perceptron.

4. After executing the file once the SVM plot are generated click on the plot to get the next plot else the program wont proceed further.

5. All the log values and result comparision are mentioned in the table in the Report file.
